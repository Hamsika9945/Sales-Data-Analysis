# Sales Data Analysis using SQL

## 📊 Project Overview
This project analyzes a sales dataset using SQL to extract meaningful business insights.

## 🔧 Tools Used
- SQL (MySQL)

## 📈 Key Analysis Performed
- Total sales calculation
- Customer and order analysis
- Regional and city-wise sales performance
- Top customers and products identification
- Monthly sales trend analysis

## 💡 Key Insights
- Identified high-performing regions and customers
- Observed monthly sales growth trends
- Found top contributing cities and categories

## 📁 Dataset
Sales dataset containing order, customer, and product details.

## 🚀 Conclusion
This project demonstrates the use of SQL for data analysis and business decision-making.
